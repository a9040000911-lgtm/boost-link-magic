## 3. Всплывающие предупреждения (⚠️ КРИТИЧЕСКИЕ ТРЕБОВАНИЯ)

PrimeLike показывает модальные окна-предупреждения при выборе определённых услуг. Это **обязательные условия**, без выполнения которых заказ не будет выполнен корректно.

### 3.1 Общие паттерны предупреждений

#### VK — Подписчики в группу (открытая группа)
> ⚠️ **Внимание!!!** Данная услуга подходит только для открытых групп. Если у Вас личная страница, воспользуйтесь услугой "Заявки в друзья". В группе должен быть открыт список участников, а также группа должна иметь хотя бы 5 постов.

**Требования:**
- Группа должна быть открытой
- Список участников должен быть открыт
- Минимум 5 постов в группе
- Для личных страниц → использовать "Заявки в друзья"

#### Instagram — Подписчики
> ⚠️ Перед накруткой обязательно отключите функцию "Пометить для проверки" иначе накрутка будет работать некорректно. Страница должна быть открыта!
> Функция "Пометить для проверки" отключается в настройках Инстаграм.
> Порядок действий:
> 1. В настройках выберите пункт "Пригласить друзей и подписаться".
> 2. Найдите опцию "Пометить для проверки" и отключите ее.

**Требования:**
- Профиль должен быть ОТКРЫТ
- Функция "Пометить для проверки" ОТКЛЮЧЕНА
- Аккаунт должен иметь аватарку (для тарифа "Стандарт")

#### Instagram — Стандарт подписчики
> Аккаунт должен иметь аватарку!

#### Telegram — Подписчики (все тарифы)
> Для открытых каналов.

**Требования:**
- Канал должен быть ОТКРЫТЫМ (публичным)
- Для закрытых каналов → использовать "Подписчики в приват канал"

#### Telegram — Подписчики для ботов
> Услуга не подходит для ботов, требующих дополнительных действий для подтверждения рефералов.

**Требования:**
- Бот не должен требовать доп. действий для подтверждения
- Достаточно простой подписки

#### Telegram — Офферные РФ/СНГ
> Для подсчета накрученных подписчиков, ссылку лучше всего указывать пригласительную.

#### Telegram — Живые пользователи
> Заказы по пригласительной ссылке могут отменяться.

#### YouTube — Премиум подписчики
> Требуется от 3 видео на канале с датой от 2-х дней. Возможны отписки.

**Требования:**
- Минимум 3 видео на канале
- Видео должны быть старше 2 дней

#### YouTube — Живые подписчики
> На канале должно быть минимум 2 видео, продолжительностью более 5-ти минут.

**Требования:**
- Минимум 2 видео на канале
- Длительность каждого видео > 5 минут

### 3.2 Типичные условия по платформам

| Платформа | Общее условие |
|-----------|---------------|
| Instagram | Профиль должен быть открыт |

| VK | Группа открыта + список участников открыт |

## 4. Структура описаний услуг

Каждый тариф содержит краткое описание из 3-6 строк:

```
Строка 1: Тип исполнителей (Боты / Боты/офферы / Живые пользователи)
Строка 2: География (Весь мир / РФ/СНГ / Разные страны)
Строка 3: Вероятность отписок (Возможны отписки / Отписки ~10% / Минимальные отписки)
Строка 4: Условие доступа (Для открытых каналов / Профиль должен быть открыт)
Строка 5: [Опционально] Дополнительное требование
Строка 6: [Опционально] Предупреждение
```

## 5. UX-паттерн предупреждений PrimeLike

### Модальное окно
- **Фон:** Яркий цвет платформы (голубой для Telegram, синий для VK, и т.д.)
- **Текст:** Крупный, белый, по центру
- **Начало:** «Внимание!!!» или «⚠️»
- **Содержание:** 2-4 предложения с конкретными требованиями
- **Кнопка:** «Принять» — пользователь подтверждает, что прочитал и понял условия
- **Когда показывается:** При выборе услуги / тарифа, ПЕРЕД заполнением формы заказа
- **Обязательное:** Пользователь не может оформить заказ, пока не нажмёт "Принять"

### Логика показа
1. Пользователь выбирает тариф
2. Если у тарифа есть `warning_text` — показывается модальное окно
3. Пользователь читает и нажимает "Принять"
4. Только после этого форма заказа становится активной

## 6. FAQ (из primelike.ru)

**Как оформить заказ?**
Выберите нужную соцсеть → услугу → тариф → введите данные → нажмите "Заказать". После оплаты заказ выполняется автоматически.

**Можно ли без регистрации?**
Да, аккаунт создаётся автоматически на указанную почту.

**Как вернуть деньги?**
Возврат на карту только если услуга не может быть предоставлена. В остальных случаях — по усмотрению администрации с комиссией.

**Есть ли скидки?**
Промокод START5 — 5% на первый заказ.

**Какая скорость?**
Зависит от тарифа. Запуск от 10 минут до 2 дней.

## 7. Ключевые особенности для реализации

1. **Поле `warning_text`** — текстовое поле в таблице `services`, содержащее HTML/text предупреждения
2. **Модальное окно** — показывается при выборе услуги с непустым `warning_text`
3. **Кнопка "Принять"** — блокирует оформление до подтверждения
4. **Цвет модалки** — наследует цвет текущей платформы
5. **Иконка ⚠️** — визуально привлекает внимание
6. **Описание в карточке** — построчный формат: тип / гео / отписки / условия
